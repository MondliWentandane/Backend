## Install dependencies

npm install --save-dev typescript ts-node nodemon @types/node @types/express @types/cors @types/bcrypt @types/jsonwebtoken @types/pg - to install pg, bcrypt, cors, express, and node

npm install --save-dev @types/bcryptjs - to install brcyptjs for user authentication

npm install pg dotenv express

npx ts-node src/server.ts
